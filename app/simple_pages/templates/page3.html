What is Pytest?

Pytest is a python based testing framework that allows users to write and execute test codes. It helps you to write simple to complex and scalable test cases for databases, APIs, or UI. Pytest is mainly used for writing tests for apis.

Advantages of Pytest:

  • Run multiple tests in parallel
  • Simple and easy syntax to learn
  • Open Source
  • Automatically detect test file and test functions automatically
  • Skip certain tests during execution

Flask Project Outline

  • Project Root Folder <- Root of the project.
    • docker-compose.yml <- Orchestration file that contains the configuration to develop locally, it overrides the Dockerfile to run the flask development server instead of running the gunicorn server that is used for hosting on Heroku.
    • Dockerfile <- File used to create an image to run in a container that runs the program.
    • heroku.yml <- Specifies the Docker image to build for the appâ€™s web process on Heroku.
    • readme.md <- A markdown file used to generate GitHub's html summary.
    • requirements.txt <- Used for specifying what python packages are required to run the project.
    • app Folder <- Contains all the files of the app.
      • __init__.py <- constructor of your package or directory and contains the Flask application object creation.
      • run.py <- This allows Gunicorn to serve the app in production.
      • simple_pages folder <- used to serve pages of the website.
        • __init__.py <- contains blueprint and routes to serve the webpages.
        • templates folder <- contains html files of the website.
          • page1.html, ... <- html pages of the website.
      • static folder <- contains assets used by the templates, including CSS files, JavaScript files, and images.
        • CSS folder <- contains css files.
        • images folder <- contains images.
        • js folder <- contains JavaScript files.
      • templates folder <- contains base.html.
        • base.html <- contains header and footer content shared amongst all the webpages.
    • tests Folder <- Contains files for testing the Flask project.
      • conftest.py <- This file makes the test configuration setup.
      • simple_pages_test.py <- Used to test the pages of the website.
